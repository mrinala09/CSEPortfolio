Home Page

Home page that leads you to sign up/login. After doing either you get redirected to main page. The main page allows you to send location and text to several options, and specify a safe area depending on the situation. This app is mainly for EHS students, staff, and parents who may need guidance during emergency procedures.

Sign-Up

Make your account: Normal (student) sign-up or Admin login. When you click sign-up, you will be prompted to either keep the given password (cake), or make your own. Will redirect them to the main page. If you try to sign-up as an admin, you will be prompted to enter a selected personal password to get the account. The password for admins is “Carr0t C^ke”. Will direct them to the admin page, which allows you to see user data.

Login

If you already have an account, click “Have an account?” to be prompted to the login page. In order to login, you will need to have done the sign-in steps already. If you signed-up as a student, you just have to enter the same username and password. You should be redirected to the main page if you typed in your information correctly. If you signed-up as an admin, type in your chosen username plus the “Carr0t C^ke” password. You can go to the main or admin page.

Main Page

If you login as a student, you will be directed here. Four Main Features:

  • Dial 911
  • Send a text to selected people
  • Indicate safe area to selected people
  • Send location to selected people
Sending your location is automatic with the other two options. Type your chosen text and chosen check boxes, then click send to deliver the message.

Admin Screen

If you login as an admin, you will be directed here. Four Main Features:

  • View User messages
  • View User data: (Username and Password)
  • Log Out
  • Go to Map to view more features
User messages will be shown for the history of the usage of this app. You will be able to view user data and admin data in separate lines for the sake of organization.
